\eslip_oauth

ESLIP OAuth

Esta clase implementa el protocolo OAuth para autenticar a un usuario intercambiando mensajes con las API's de los Proveedores de Identidad. Esta clase soporta las versiones 1.0, 1.0a y 2.0 de OAuth.

GetUserData()

GetUserData() : array

Realiza una peticion de los recursos del usuario a la API Oauth del proveedor de identidad con el que se esta interactuando. Para poder realizar este pedido previamente se debe haber obtenido el token de accesp correspondiente La URL a la que se debe comunicar, y los parametros que se utilizan fueron previamente obtenidos y configurados en el plugin. Dependiendo de la version del servidor OAuth, será el tratamiento que se le realizarán a los parametros para luego ser enviados.

Throws

\EslipException

Si el servidor OAuth del proveedor de identidad posee una version no soportada por el plugin

\EslipException

Si todavia no se obtuvo el access token para poder hacer el pedido de recursos

Returns

array —

Arreglo con los recursos del usuario devueltos por el servidor OAuth

Tags

SendAPIRequest()

SendAPIRequest(string $url, string $method, array $parameters, array $oauth, array $options) : array

Realiza una peticion a la API Oauth del proveedor de identidad con el que se esta interactuando.

La URL con la que se desea comunicar, y los parametros que se utilizan son recibidos por este metodo por parametro. Dependiendo de la version del servidor OAuth, será el tratamiento que se le realizarán a los parametros para luego ser enviados. La conexion con el servidor OAuth se realiza utlizando la libreria cURL.

Parameters

string $url

URL de la API OAuth con la que nos vamos a comunicar
string $method

Metodo HTTP que se va a utilizar para comunicarse con el servidor
array $parameters

Parametros que se van a enviar en la petición
array $oauth

Parametros especificos de OAuth 1.0 que se van a enviar en la petición
array $options

Informacion adicional para la interfaz o manejo de errores

Returns

array —

Arreglo con los datos devueltos por el servidor OAuth
